# 标准游戏资讯 cube 模板，2*n图 + 12图文内容

## 介绍
标准游戏资讯模板

## 使用
```
cubetool init <cubeid> -t https://github.com/ntt360/cube-tpl-game-2pic12pictext.git
```
或简写：
```
cubetool init <cubeid> -t ntt360/cube-tpl-game-2pic12pictext
```
